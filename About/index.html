GOTO QUIZ

About Us

The CodeTrivia application was developed as part of a project for CSCI 4440 - Software Design & Documentation at RPI by Anthony Bonadies, Eric Zhang, James Gao, Sitong Liu, and Harrison Riley.

This application is meant to provide trivia quizzes based on code categories and allow users to do quizzes and compete for high scores with each other while learning about code in an engaging way.

Quiz Rules

You will be presented with a certain number of questions based on the category they select. Questions will then scale in difficulty as they go on, but will offer more points for more difficult questions. You will be given tools such as a half tool to half the choices on a question and a choice tool to be provided with a choice for an answer.

Contact Information

If there are any issues or suggestions voice your opinions to codetriviadevteam@gmail.com. The team will provide other methods of contact in the future as well such as Discord if people cannot voice their opinions through email.
